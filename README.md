## Project Description
This is a console-based Cafeteria Counter Management System in C++.  
It manages orders, inventory, payments, and customer feedback.  

## How to Run
1. Compile all `.cpp` files together in your IDE (e.g., Code::Blocks, Dev-C++).  
2. Run `main()` to start the system.  
3. Navigate the menus to place orders, manage inventory, and view total sales.
